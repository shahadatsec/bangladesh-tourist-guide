# Bangladesh Tourist Guide

A beginner Android application built with **Java**, **XML**, and **Glide**. This project showcases some of the most popular tourist destinations in Bangladesh with images loaded from the internet.

## Features

* 🏖️ Home screen with a simple and clean UI
* 🖼️ Tourist places displayed in a ScrollView
* 🌐 Images loaded from URLs using Glide
* 🎨 Simple and beginner-friendly Material-style design
* 📱 Built for Android Studio

## Tourist Places

* Cox's Bazar
* Sajek Valley
* Sundarbans
* Saint Martin
* Srimangal

## Technologies Used

* Java
* XML
* Glide
* Android Studio
